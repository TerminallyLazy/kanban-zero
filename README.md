# Kanban Zero

AI-native, energy-aware task management for ADHD brains.

## Project Structure

- `backend/kz/` - FastAPI backend with AI-powered task parsing
- `cli/kz/` - Typer CLI for terminal task management
- `web/` - Next.js frontend (coming soon)

## Setup

```bash
# Install dependencies
uv sync

# Start development
./scripts/dev.sh
```
